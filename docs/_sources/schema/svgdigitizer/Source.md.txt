## Slots

| Name | Cardinality and Range | Description | Examples |
| ---  | --- | --- | --- |
| [citationKey](citationKey.md) | 1 <br/> [String](String.md) | Unique citation key for bibliographic reference, usually <FirstAuthorName>_<Y... | `electrobear_2024_electrochemistry_563` |
| [url](url.md) | 1 <br/> [HttpUrl](HttpUrl.md) | DOI (URL) to the publication or dataset | `https://doi.org/10.1234/example.2024` |
| [techniques](techniques.md) | * <br/> [String](String.md) | Experimental techniques used in the study | `CV`,  `EIS`,  `CA` |
| [figure](figure.md) | 0..1 <br/> [String](String.md) | Figure number or identifier from the source including subfigure label | `Figure 2a` |
| [originalFilename](originalFilename.md) | 0..1 <br/> [String](String.md) | Original filename of the data file | `2023-04-24_cv_data_001.csv` |
| [bibdata](bibdata.md) | 0..1 <br/> [String](String.md) | BibTeX bibliographic data | `@article{electrobear_2024_electrochemistry_563, title={...}, ...}` |
| [comment](comment.md) | 0..1 <br/> [String](String.md) | Additional notes about the source | `The data in the figures of this work are blurry.` |
| [curve](curve.md) | 0..1 <br/> [String](String.md) | Specific curve or trace identifier within a figure | `blue`,  `solid`,  `1` |
